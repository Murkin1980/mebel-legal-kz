# PRODUCT.md — MebelLegal KZ

## Продуктовое видение

MebelLegal KZ — правовой контур для мебельного бизнеса Республики Казахстан.

Система помогает мебельным компаниям:

- подготовить юридически корректный комплект документов заказа;
- минимизировать правовые риски через проверку по утверждённым источникам;
- обеспечить прослеживаемость каждого юридического решения;
- получить согласование клиента на конкретную версию документов;
- контролировать изменения заказа и их правовые последствия.

## Целевая аудитория

- Мебельные производства (ТОО, ИП);
- Юристы, обслуживающие мебельный бизнес;
- Менеджеры, ответственные за договорную работу.

## Проблема

Мебельный бизнес в Казахстане работает с:

- стандартными договорами, которые не всегда учитывают специфику;
- отсутствием системного контроля за изменениями заказов;
- рисками из-за использования устаревших формулировок;
- сложностью отслеживания, какая версия договора согласована клиентом.

## Решение

MebelLegal KZ создаёт управляемый правовой контур:

1. **Реестр правовых источников** — утверждённые редакции законов и подзаконных актов.
2. **Правила проверки** — детерминированные проверки договоров по источникам.
3. **Версионные шаблоны** — каждый шаблон имеет статус, версию и историю.
4. **Юридический кейс** — контейнер правового процесса одного заказа.
5. **Договорный комплект** — версия договора с привязкой к правилам и источникам.
6. **Согласование клиента** — доказуемое подтверждение конкретной версии.
7. **История изменений** — неизменяемый audit log всех действий.

## Модель.stage 1 — Текущий этап

На первом этапе реализуется:

- мультитенантное ядро (организации, роли, RLS);
- минимальная модель юридического кейса;
- доменные команды с проверкой прав;
- append-only audit log;
- идемпотентность операций;
- деньги в тийинах.

**Не реализуется на этапе 1:**

- подключение ИПС «Әділет»;
- AI/LLM функционал;
- генерация документов;
- публичное согласование;
- интеграции с MebelDocs AI и Interactive KP.

## Ключевые метрики успеха

- данные разных организаций строго изолированы;
- все изменения проходят через проверяемые команды;
- юридически значимая история не перезаписывается;
- модель готова к масштабированию.

## Конкурентные преимущества

- фокус на специфике мебельного бизнеса РК;
- привязка к утверждённым правовым источникам;
- прослеживаемость каждого юридического решения;
- мультитенантность с полной изоляцией данных.
